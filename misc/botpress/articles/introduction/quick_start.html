Quick start

The purpose of this guide

This guide is intended to explain how to quickstart using your own AI chatbot from Botpress.

It provides troubleshooting assistance each step of the way for dealing with common issues.

If your questions are left unanswered at any point in this guide, message the automated troubleshooting representative via the chat icon in the bottom right corner.

If you still haven't found an answer by the end of the guide, contact Botpress' team through their Discord support center

What this guide does

By following this guide, you will learn how to build your own Botpress chatbot that can generate images with AI.

Once the bot is complete, you will learn how to publish it, share it, and put it on your own website.
