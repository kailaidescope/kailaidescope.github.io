Instructions

Introduction

This guide assumes that you are hosting your website from a raw HTML file, along with optional CSS and Javascript.

Depending on how you are hosting your website the steps for embedding the webchat may vary. If this guide is not applicable, see the other guides in Ways to embed.

Finding embedded scripts

To embed your chatbot into your website's raw HTML, all you have to do is copy two script tags into your HTML file.

You can find these script tags by going to the Dashboard > select your bot from the left sidebar > Webchat > Share.

Embedding scripts

Copy the style tags in the Embed code section and paste them into your raw HTML file just above the closing </body> tag. For more information on the embedding process, you can follow this embedding guide.

Once the two <script> tags are added, you can save your HTML file and update your website hosting platform to use the updated HTML file.

Then, the embedded scripts will inject the chatbot's UI into your page. The bot will be accessible via a chat button in the bottom right corner of your website.

If you want to style your chat pane, check out how you can style the chat.

Troubleshooting

I've embedded the script, so why can't I access the chat?

The most common issues are resolved through these steps:

  1. Try reloading the page
  2. Ensure the edited HTML file is properly saved to the hosting service you are using and reload the page
  3. Ensuring the script tag is positioned in the right place, above the closing </body> tag
  4. Ensure the copied code is correct
  5. Ensure the <script> tags are properly closed with a </script> closing tag

If the above solutions do not resolve the problem, there may be a problem with the script injection process.

Ensure that the script tag is located in the main HTML file you are loading, rather than an HTML file injected into another at runtime (like a modular component). Since the Webchat script is called upon load, this should fix loading errors.

If you are still having trouble, talk to our automated troubleshooting bot or contact Botpress' Discord support.

How can I style the chat?

For control over your chat's styling, you can go to the Dashboard > select your bot from the left sidebar > Webchat > Theme.

The Styles section allows you to write CSS that affects how your bot's chat panel is displayed. You can also choose the primary color of your chat pane under the Theme header.

Why is the chat formatted strangely?

You may have made changes to the CSS styling of the chat pane. To revert to the original style, delete any CSS in the Dashboard > select your bot from the left sidebar > Webchat > Theme menu.

If this is not the error, it is possible, but unlikely, that specific styling on your website could impact the chat's format. Try inspecting your website's CSS to find any bugs in the styling.
