Accessing workflows

Instructions

Enter the studio for the bot you want to build a workflow on. If the bot has not gone through the basic setup process, complete that first.

On the left sidebar, select the Workflows tab, if not already selected. The Workflows tab looks like this:

You will see the pre-configured workflows: Main, Error, Timeout, and Conversation End. You can edit these by clicking on them, or make a new workflow by clicking Create Workflow.

Troubleshooting

How can I see the pre-configured workflows?

You should always be able to see the Main, Error, Timeout, and Conversation End workflows. If you can't see them, try refreshing the page or closing and re-opening your browser.

Further issues

If you are still experiencing issues, talk with our automated troubleshooting representative via the chat icon in the bottom right corner, or contact Botpress' team through their Discord.
