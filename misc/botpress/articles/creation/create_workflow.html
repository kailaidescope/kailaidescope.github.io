Create a new workflow

Instructions

Setup a new workflow

Once in the workflows tab, select Create Workflow, and give it a descriptive name.

Delete the Standard node connected to the Entry and Exit nodes. Then replace it with an AutonomousNode by secondary clicking the dotted background and clicking AutonomousNode.

Attach the backgroundAutonomousNodes' entry connection point to the Entry node and the AutonomousNodes' exit connection point to the Exit node.

Setup image generation in your workflow

Allow your bot to generate a picture on command by finding the AutonomousNode in your new workflow and clicking Add Card.

A list of Cards will pop up, from which you can search "Generate Image."

Click on the card labeled Generate Image. It will automatically be added to your AutonomousNode.

To give special instructions for image generation, click on the Generate Image card in the AutonomousNode and write in the Instructions field.

With the card added, your AutonomousNode can choose to generate an image when prompted. Since this node is not yet connected to the Main workflow, your bot can't use it during conversation yet.

Make your bot generate images

Let the bot use your new workflow by connecting it to the Main workflow.

Start by opening the Main workflow, secondary-clicking the dotted background, and selecting the Execute a Workflow node with your new workflow's name attached.

Hook up the Main workflow's Start and End nodes to the Execute a Workflow node.

Your bot is now ready to be tested! To test the bot's new functionality, open the Emulator chat on the right side of the screen. It may already be open.

Write a prompt like: "Hi! Can you generate an image of a cat for me?" The bot will take a moment to process, then will send back a generated image.

Troubleshooting

How can I create a new workflow?

To create a new workflow, first navigate to the workflows tab on the top left sidebar of the studio.

Above the list of existing workflows, click Create Workflow, and a new workflow will be initialized for you.

How can I add a node?

To add a node to your bot, first navigate to the workflow you want to put it in. If you are unsure which workflow to edit, choose the Main workflow.

When you see the workflow's nodes displayed, secondary click on the dotted background to bring up the node creation menu.

Select any node you want, feel free to experiment with different kinds of nodes to see what they do.
