Modify a workflow

Note: Since a basic workflow setup is provided upon initialization, modifying the Main workflow is the easiest way to quickstart your bot.

Instructions

Modify Main workflow

Once in the workflows tab, select the Main workflow to configure the core conversational flow of your bot.

To allow your bot to generate a picture on command, find the AutonomousNode in the Main workflow and click the Add Card button within the AutonomousNode.

A list of cards will pop up, from which you can search "Generate Image."

Click on the card labeled Generate Image. It will automatically be added to your AutonomousNode.

To give special instructions for image generation, click on the Generate Image card in the AutonomousNode and write in the Instructions field.

With the card added, your AutonomousNode can choose to generate an image when prompted. Since this node is in the Main workflow, your bot will automatically use it during conversation.

Test the bot

To test the bot's new capabilities, open the Emulator chat on the right side of the screen. It may already be open.

Write a prompt like: "Hi! Can you generate an image of a cat for me?" The bot will take a moment to process, then will send back a generated image.

Troubleshooting

Where can I find the Main workflow?

If you aren't in the workflows tab, navigate to the top left sidebar of the studio, then click on the top button to access the workflows tab.

If you are already on the workflows tab, you may have to expand the list of workflows by clicking the Show Explorer button on the right of the workflows tab button.

Once the workflow list appears, the Main workflow will be at the top of the list, marked by a green flag.

Where can I find the AutonomousNode?

To access the auto-generated AutonomousNode in the Main workflow:

  1. Navigate to the workflows tab
  2. Click on the Main workflow
  3. The AutonomousNode will be connected to the Start and End nodes
  4. You can change it by clicking on its cards or its Instructions box

To create a new AutonomousNode:

  1. Navigate to the workflow in which you want to add the node
  2. Secondary click on the dotted background of the work area
  3. Select AutonomousNode

How can I connect nodes?

You can connect nodes by using their connection pointsâ€”the little grey dots at the top left and bottom right of a node.

Connection points on the left side are entry points and connection points on the right side are exit points. Node A is activated when another node B finishes working, if B's exit point is connected to A's entry point.

Click and drag from one node's connection point to another node's connection point to create a link between the nodes.

How can I delete a node?

Locate the node you want to delete and secondary click on it, then select Delete Node.

How can I delete a card?

Locate the card you want to delete and secondary click on it, then select Delete Card.

Why do I get an error when generating an image?

If you are seeing an error message in the emulator:

  • These error messages are not always fatal, sometimes the bot needs to try a request twice
  • These error messages won't be displayed to the user in the published version, but the bot may tell the user if an error was encountered
  • If the error is fatal, check your bot's logic
  • If nothing seems wrong, try refreshing the studio page

Why can't my bot generate an image?

There are several reasons why your bot might not generate an image:

  • If you put a Generate Image card into an AutonomousNode, the bot decides when to generate an image. Try explicitly prompting your bot to generate an image, or add instructions to the bot or its Generate Image card to tell it when to generate images
  • If prompting isn't the problem, check if you have exceeded your price plan limit. If you have, consider upgrading to a higher price limit
  • If you haven't put a Generate Image card into your AutonomousNode, try adding one, republishing your bot, and prompting it for an image again
  • If it still doesn't work, try creating a StandardNode and putting a Generate Image card inside it. Make sure you fill in the required prompt and ID fields. If this doesn't work, contact Botpress' support on their Discord.

Further issues

If you are still experiencing issues, talk with our automated troubleshooting representative via the chat icon in the bottom right corner, or contact Botpress' team through their Discord.
